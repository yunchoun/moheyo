CardContainer 컴포넌트 JSX return ( <div> <Card color={"#ff6663"}></Card> <Card color={"#ffa737"}></Card> </div> ); 4. Card 컴포넌트 JSX return ( <div className="card"> <Square color={color}></Square> <Label color={color}></Label> </div> ); 5.
